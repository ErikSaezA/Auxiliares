| ------- | -------------------- |

| < 1.0   | :x:                  |

## Reportar una Vulnerabilidad

Si encuentras una vulnerabilidad de seguridad en **Auxiliares - Recursos Educativos Abiertos**, por favor sigue estos pasos para reportarla:

1. **Envío del Reporte**: Envía el reporte de vulnerabilidad a [erik.saez@ug.uchile.cl](mailto:erik.saez@ug.uchile.cl) o abre un **Issue** privado en este repositorio.
2. **Respuesta Inicial**: Recibirás una confirmación de recepción de tu reporte dentro de las 24 horas siguientes.
3. **Evaluación y Actualización**: El equipo de desarrollo evaluará la vulnerabilidad. Durante este proceso, puedes esperar actualizaciones cada 3 días sobre el estado del reporte.
4. **Resolución**: Si la vulnerabilidad es aceptada, se proporcionará una corrección o actualización en la siguiente versión disponible. Si la vulnerabilidad no es aceptada, se te informará de los motivos por los cuales no se abordó.
5. **Divulgación Responsable**: En caso de que se acepte y se resuelva la vulnerabilidad, se realizará una divulgación responsable de la solución implementada.

Gracias por ayudar a mantener el proyecto seguro para todos los usuarios.
